Project Overview

About the Game

Final Bell is a single-player survival horror game created using Unreal Engine 5. The game takes place in a zombie-infested dormitory environment where the player must explore, survive enemy encounters, and manage available resources.

The project focuses on creating a tense survival experience through exploration, enemy AI, melee combat, inventory management, health items, and a flashlight mechanic.

Challenges & Solutions

01

Enemy Navigation & Movement

Implementing enemy movement and ensuring zombies could navigate the environment correctly was one of the challenges during development.

SOLUTION

I used Unreal Engine's NavMesh navigation system and adjusted the level setup and enemy logic to improve navigation and player pursuit.

02

Character Animation Integration

Integrating external character animations into the project required adapting animations to work correctly with the game characters.

SOLUTION

I worked with animation retargeting in Unreal Engine to integrate Mixamo animations and adapt them for use with the project's character setup.

03

Creating the Horror Atmosphere

Creating a dark environment while maintaining enough visibility for gameplay required balancing the lighting and atmosphere.

SOLUTION

I experimented with Unreal Engine lighting, post-processing, environmental effects, and the flashlight mechanic to create a darker atmosphere while keeping the environment playable.

06

What I Learned

Developing Final Bell gave me hands-on experience with building and integrating multiple gameplay systems in Unreal Engine 5. Through this project, I improved my understanding of Blueprint scripting, enemy AI and navigation, player mechanics, inventory systems, animation integration, and level design.

The project also taught me how different gameplay systems need to work together to create a complete experience. Working through technical and design challenges helped strengthen my problem-solving skills and gave me a better understanding of the overall game development workflow.
